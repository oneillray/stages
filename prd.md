# PRD: Indoor Cycling Zone Calculator (Mobile-First)1. 

## Core Concept 

A high-performance, mobile-first web app for indoor cyclists to reference their training zones. The app operates entirely client-side, using local storage for persistence and a tabbed navigation system for a native-app feel.2.

- Framework: React (Vite) or Next.js (Static Export).Styling: Tailwind CSS.Animations: framer-motion (installed for future-proofing and section transitions).Icons: lucide-react.Font: IBM Plex Sans.Storage: Browser localStorage for Metrics (FTP, MaxHR, Weight) and History.3.
